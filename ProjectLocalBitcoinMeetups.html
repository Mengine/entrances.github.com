WYSIWYG page in that it contains pure text, to save time from fiddling with tags and formatting 
and also for making it easier to edit.

update this page by clicking edit data above and entering your details below. Pros: The page size 
limit is almost unlimited. Also, it's very quick for me to edit. Cons: You need a Github account to 
edit this page. (But they are easy to get if you don't already have one).
